# 🎮 Compro Game

A browser-based character selection and quiz battle game built with **HTML**, **CSS**, and **JavaScript**.

> 🧩 This repository contains the **source code** of the Compro Game.  
> 🌐 The live site was previously hosted at [https://compro-game.vercel.app/](https://compro-game.vercel.app/), but is **no longer active**.

---

## 🕹️ Overview

Compro Game is an interactive web project inspired by the fun of coding battles and creative design.  
Players select a character, answer coding questions, and compete in a “battle” based on performance.

The project demonstrates:
- DOM manipulation  
- CSS animation and transitions  
- Event-driven JavaScript logic  
- Simple game logic based on correct/incorrect answers  

---

## 💡 Inspiration

This project was inspired by the **ComPro (Computer Programming)** course theme — blending coding education with playful design.  
It serves as an **introduction to developing interactive browser-based applications**.

---

## ⚙️ Features

- **Character selection** with hover animations  
- **Dynamic quiz system** for battle logic  
- **Win/Loss animations** (victory and defeat screens)  
- **Responsive UI** using HTML and CSS  
- **Custom transitions** with `@keyframes`  

---

## 🧠 How It Works

1. Choose a character from the selection screen.  
2. Answer coding questions correctly to power up your character.  
3. The outcome (win/lose) is based on your answers.  

---

## 🛠️ Technologies Used

- **HTML5** – Structure and layout  
- **CSS3** – Styling, transitions, animations  
- **JavaScript (Vanilla)** – Logic, interactivity, and state handling
